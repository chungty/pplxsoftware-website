Getting Started

Install from Atlassian Marketplace. Works in 60 seconds, zero configuration.

Apps > TeamOps > Team Directory > Add Employee.

Apps > TeamOps > Onboarding > Start Onboarding > pick a template.

Apps > TeamOps > Offboarding > select a template > enter last day.

Your Jira admin can upgrade from the Atlassian Marketplace billing page.

Security & Administration

All data is stored in Atlassian's Forge infrastructure, encrypted at rest, following your organization's data residency settings. PPLX Software has zero access to your data. No employee information leaves Atlassian's cloud.

Your data is stored in Forge's Custom Entity Store, which is managed by Atlassian. Uninstalling removes the app UI but data retention follows Atlassian's Forge data policies. Contact support if you need data exported before uninstalling.

TeamOps requires: storage:app, read/write:jira-work, read:jira-user, read:group:jira, manage:jira-project, and manage:jira-configuration. Each scope is documented in our security page.
